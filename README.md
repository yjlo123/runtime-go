# Runtime Go
The Runtime Script programming language written in Golang

Learn more about Runtime Script:  
https://github.com/yjlo123/runtime-script

## Usage  
```shell
$ runtime <file_path>
```

Commands NOT supported:  
```
clr
drw
pxl
```
